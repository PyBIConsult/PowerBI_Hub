# COALESCE
Coalesce is an interesting and powerful function. 
It takes two arguments (or measures), it returns the first argument, 
if the result of the first argument is blank, it returns the second argument

```c#
COALESCE = 

// expression
COALESCE( 
  // first argument
  [measure1], 
  // second argument
  [measures2] 
)
```
